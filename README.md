# react-bonobo

React renderer to build UI interfaces using canvas/WebGL (TV)

## Installation

```sh
npm install react-bonobo
```

## Usage

```js
import { multiply } from 'react-bonobo';

// ...

const result = await multiply(3, 7);
```
